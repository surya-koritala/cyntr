Everything you need to run AI agents in production

5 LLM Providers

Claude, GPT, Gemini, Ollama, and any OpenAI-compatible API. Switch providers per agent.

7 Messaging Channels

Slack, Teams, WhatsApp, Telegram, Discord, Email, Webhook. Cross-channel identity binding.

8 Agent Tools

Shell, HTTP, file operations, web browser, GitHub, Jira. Agents can take real actions.

Policy Engine

Deny-by-default YAML rules. Per-tenant, per-agent, per-tool granularity. Human-in-the-loop approvals.

Audit Logging

Tamper-evident SHA-256 hash chains. Ed25519 signing. Daily rotation. SOC 2, GDPR, HIPAA ready.

Multi-Tenant

Isolated agents, policies, and audit trails per team. Namespace, process, or Docker isolation.

Workflow Engine

Chain agent actions with conditions, retries, and webhooks. Automate multi-step processes.

Federation

Multi-site deployment with policy sync and cross-site audit queries. No central controller.

Single Binary

One go build, one binary. No Docker required. No external databases. Just run it.
